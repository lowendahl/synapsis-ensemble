This is a base truth — voices may reference but not modify.

Umbrella: **Symbiont** (internal codename, no trademark filing). Externally
always qualified with **CSU**, e.g. "CSU · Symbiont Ensemble".

## Naming rules

- Surface name + middle dot `·` (U+00B7) for the formal qualifier.
  ✅ `CSU · Symbiont Ensemble`     ❌ `CSU - Symbiont Ensemble`
- Eyebrow microcopy: `CSU · SYMBIONT` — ALL CAPS, letter-spacing 1.5–2px.
- Tagline (Ensemble): **Many voices. Many instruments. One workspace.**
- Tagline (Chorus): **Many voices. One passage.**

above are for the wordmark, eyebrow, anchor dot, and brand assets only.

## Typography

- **Wordmark:** Source Serif 4 (display)
- **Eyebrow / UI labels:** Segoe UI Variable, ALL CAPS, +1.5–2px letter-spacing
- **Body / UI:** Segoe UI Variable, regular

## Mark

Vertical line + 3 asymmetric curves + anchor dot = "held resonance / many
voices around one anchor". Ensemble uses the same mark with a 4-curve variant
in headers (one curve per voice family).

## Voice & tone

- Concrete, specific, never aspirational filler.
- "Many voices. Many instruments. One workspace." — held tension between
  multiplicity and unity. Avoid "AI", "agentic", "powered by". Speak like a
  workshop, not a demo.
- Anti-patterns: "supercharge", "unleash", "AI-first", "synergy".
