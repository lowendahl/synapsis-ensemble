# Components

Reusable UI patterns documented here. Source of truth is `renderer/index.html` (single-file by design until v0.4); this folder just describes intent + states.

## Inventory

- `voice-tab.md` — the per-voice tab in the strip (active, inactive, with-ping)
- `target-chip.md` — the council selector dot (selected, unselected)
- `instrument-card.md` — the right-rail per-voice tool card
- `base-truth-card.md` — base-truth reference card with LOCAL/EXTERNAL/MISSING pill
- `pane.md` — the generic content pane shell (header / body / input)
