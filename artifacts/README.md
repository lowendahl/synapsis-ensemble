# Artifacts

Live working artifacts. Drafts, screenshots, exports, things-in-progress.

This folder is **noisy on purpose**. Nothing here is canonical. When something stabilizes, promote it to `product-design/`, `brand/`, or `decisions/`.

## Conventions

- Date-prefix volatile files: `2026-05-07-tab-scroll-mockup.png`.
- Don't put anything here you'd be sad to lose.
- Subfolders OK. No required structure.
