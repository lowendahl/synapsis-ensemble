# Precedence — Synapsis · Ensemble

When sources disagree, follow this order (highest to lowest authority):

1. **`base-truths/symbiont-brand.md`** — Symbiont rebrand decisions are
   externally locked. Never overrule without an explicit rebrand sign-off.
2. **`clawpilot/conventions.md`** — stack/architecture rules. Breaking these
   needs a `memory.md` entry explaining why.
3. **`clawpilot/context.md`** — scope and "what we're building". Disagreements
   here usually mean the user has redirected scope; update `context.md` rather
   than fight it.
4. **`clawpilot/memory.md`** — running decisions. Latest entry wins over
   earlier ones.
5. **The user's current message.** If it contradicts memory, ask before
   overruling — they may want to update memory, or they may have forgotten.
6. **Inferred best practice.** Last-resort default.
