# Ensemble — Brand

This is the **product brand** for Ensemble specifically. The umbrella brand (Symbiont) is a base-truth and lives at `../base-truths/symbiont-brand.md`.

## Files

- `voice-and-tone.md` — How Ensemble talks (UI copy, error messages, doc tone).
- `naming.md` — Why "Synapsis · Ensemble", what's a Voice, what's the Council, what's an Instrument.
- `palette.md` — Token usage. Anchored on Clawpilot's `--cp-*` variables — Ensemble does not invent its own palette.
- `references/` — Inspiration moodboard. External references only (links + short notes).

## Tied to base-truths

- `../base-truths/symbiont-brand.md` is the **upstream**. If something here contradicts the Symbiont locked decisions, the base-truth wins.
